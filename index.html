Proportional Columns and Rows Using Fr

The container is divided into three columns, where the middle column takes up twice the available space compared to the other two columns. There are two rows, with the second row taking up three times the space of the first row.

Mixing Units for Columns and Rows

This code creates a grid container with three columns: the first column has a fixed width of 150px, the second column takes up the remaining available space, and the third column takes up 20% of the container's width. The container has two rows, each taking up 50% of its height.

Using the span Keyword

This code creates a grid container with three columns of 200px, 300px, and 150px width, and two rows of 100px and 200px height. The first item of the grid-container is set to take up two columns and two rows by using the span keyword with a value of 2 set to the grid-column property and grid-row property respectively.

                        .grid-section__container-span-keyword {
                        display: grid;
                        grid-template-columns: 200px 300px 150px;
                        grid-template-rows: 100px 200px;
                        grid-gap: 10px;
                        }

                        .grid-item--big3 {
                        grid-column: 1 / span 2;
                        grid-row: 1 / span 2;
