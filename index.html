Proportional Columns and Rows Using Fr

The container is divided into three columns, where the middle column takes up twice the available space compared to the other two columns. There are two rows, with the second row taking up three times the space of the first row.

.grid-section__container-fr {
display: grid;
grid-template-columns: 1fr 2fr 1fr;
grid-template-rows: 1fr 3fr;
gap: 10px;

Grid Areas

This code creates a grid container with three columns and four rows, and specifies the grid-template-areas property to divide it into areas. It also sets the grid-area property for each item to specify where it should be placed.

.grid-section__container-areas {
display: grid;
grid-template-columns: repeat(3, 1fr);
grid-template-rows: repeat(4, 1fr);
gap: 10px;
grid-template-areas:
"header header header"
"news news aside"
"promo promo aside"
". footer footer";
}

.header {
grid-area: header;
}

.news {
grid-area: news;

.promo {

Controlling Grid Item Placement with grid-auto-flow

This code creates a grid container with a width of 500px and three rows, and sets the grid-auto-flow property to column. This means that new grid items will be placed in columns, instead of rows, when the available space in the current row is filled. The order of the items will be based on the source order.

.grid-section__container-auto-flow {
    display: grid;
    width: 500px;
    grid-template-rows: repeat(3, 100px);
    grid-auto-flow: column;
    gap: 10px;
}

Controlling Grid Item Placement with grid-auto-flow: dense

This code creates a grid container with a width of 500px and three rows, and sets the grid-auto-flow property to dense. This means that new grid items will be placed based on available space, filling any gaps left by previous grid items. The order of the items may not follow the source order to fill gaps and minimize empty spaces.

.grid-section__container-auto-flow-dense {
    display: grid;
    width: 500px;
    grid-template-rows: repeat(3, 100px);
    grid-auto-flow: dense;
    gap: 10px;
}
